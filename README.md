## 使用教學

1. 先上Github 把檔案抓下來點擊Code
2. 並下載檔案Download ZIP
3. 將解壓縮後將zh-TW資料夾放入ASM裡的Globalization資料夾
4. 重啟即可在選項看到中文-台灣
